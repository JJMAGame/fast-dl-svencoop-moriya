<h1>FastDL do SC-Moriya</h1>

<b>O que é o FastDL</b>
refere-se principalmente a um método para otimizar o download de arquivos de servidores de jogos, como mapas, modelos e texturas customizadas, para que os jogadores recebam o conteúdo mais rapidamente ao entrar no servidor, evitando lentidão.
e todos os conteúdo tirado daqui também tem de outros github
